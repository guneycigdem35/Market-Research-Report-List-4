<p><ul><li>Half Wave Dipole</li><li>Quarter Wave Dipole</li><li>Uniform Linear Array Antenna</li><li>Non-Uniform Array Antenna</li></ul></p>
<p><p>Radar antennas have various types including half wave dipole, quarter wave dipole, uniform linear array antenna, and non-uniform array antenna. Half wave dipole antennas are known for their simplicity and efficiency. Quarter wave dipole antennas are smaller in size and widely used in portable devices. Uniform linear array antennas consist of multiple identical elements arranged in a straight line for higher directivity. Non-uniform array antennas have elements with varying spacing to achieve specific radiation patterns. Each type of antenna serves different purposes in radar applications.</p></p>
